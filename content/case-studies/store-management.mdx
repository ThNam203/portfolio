## Overview

A retail-store management dashboard covering inventory, point-of-sale, staff, customers, and reporting. Built as a Next.js frontend on top of a Spring Boot REST backend.

## Stack

Next.js (App Router), TypeScript, Spring Boot, Java, PostgreSQL, JWT auth.

## Highlights

- **Multi-module dashboard.** Inventory, POS, staff, customers, reports — each module fully separable.
- **Role-based access.** JWT-based auth with role claims gating UI and API.
- **Paginated heavy lists.** Server-side pagination + filtering on the Java backend to keep large catalogs snappy.
- **Reporting.** Aggregations rolled up in SQL, not at the application layer.

## What I'd do differently

- Replace REST polling on the dashboard with server-sent events for live metrics.
- Push more business logic into typed contracts (OpenAPI → TS) instead of hand-typing the client.
