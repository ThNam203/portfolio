## Tổng quan

Dashboard quản lý cửa hàng bán lẻ phủ kho, POS, nhân sự, khách hàng, báo cáo. Xây dưới dạng frontend Next.js trên backend Spring Boot REST.

## Stack

Next.js (App Router), TypeScript, Spring Boot, Java, PostgreSQL, JWT auth.

## Điểm nổi bật

- **Dashboard đa module.** Tồn kho, POS, nhân sự, khách hàng, báo cáo — mỗi module hoàn toàn tách rời.
- **Phân quyền theo vai trò.** JWT auth với role claim gate UI và API.
- **List lớn phân trang.** Phân trang + filter server-side trên backend Java giữ catalog lớn vẫn mượt.
- **Báo cáo.** Tổng hợp roll-up trong SQL, không ở tầng application.

## Điều tôi sẽ làm khác

- Thay REST polling trên dashboard bằng server-sent event cho metric live.
- Đẩy thêm business logic vào contract có kiểu (OpenAPI → TS) thay vì tự type client.
